# CRUD Application Using JSP, Servlet & JDBC

## 📌 Project Overview

This project is a simple CRUD (Create, Read, Update, Delete) web application developed using JSP, Servlet, JDBC, HTML, CSS, and MySQL. The application allows users to manage records efficiently through a user-friendly interface.

## 🚀 Features

* Create new records
* Read/View all records
* Update existing records
* Delete records
* Responsive Navigation Bar
* Form Validation
* Database Connectivity using JDBC

## 🛠️ Technologies Used

* Java
* JSP
* Servlet
* JDBC
* MySQL
* HTML5
* CSS3
* Apache Tomcat

## 📂 Project Structure

* index.jsp → Home Page
* navbar.jsp → Navigation Menu
* form.jsp → Add New Record
* read.jsp → Display Records
* update.jsp → Update Existing Record
* Servlet Classes → Business Logic
* DAO Classes → Database Operations
* MySQL Database → Data Storage

## ⚙️ CRUD Operations

### Create

Users can add new records using the form page.

### Read

All records are displayed in a tabular format.

### Update

Existing records can be modified through the update page.

### Delete

Users can remove records from the database.
